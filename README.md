
# Nest.js Noserver Pokemon Collector

Nest.js standalone application (without server) collect pokemons from [pokeAPI](https://pokeapi.co/)

The applicaiton basod on [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
  npm install
```

## Running the app

```bash
  # development
  npm run start

  # watch mode
  npm run start:dev

  # production mode
  npm run start:prod
```

## Test

```bash
  # unit tests
  npm run test

  # e2e tests
  npm run test:e2e

  # test coverage
  npm run test:cov
```
